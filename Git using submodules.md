# Git using submodules functions

To use git submodule is simple, if you 
do notice your orders.

```
git submodule add <url> <local-path>
```

Yes, that's simple but it's not like .gitignore you 
can just add it you want.

Git sub-modules working will store records by ".gitmodules" 
but you can't create this file mannually...

You should only use command provided by git to use sub-modules
functions.

If unessarry, just create a full project.

But it might be confusing when you working a project depends 
others (little whisper) :(
